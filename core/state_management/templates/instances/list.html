新增 MongoDB 實例提示

  • 建議為每個實例命名（如「Default」、「QA-replica」），方便在路由規則 mongodb_instance 欄位引用。
  • 欄位 host/port/database 必須對應實際資料庫，並確保帳密可被 Web UI 與分析節點使用。
  • 僅「啟用」的實例會由 TaskScheduler 監看新資料，上傳來源需位在所選集合（預設 recordings）。
  • 新增或修改後請使用列表右側的插頭圖示測試連線。
